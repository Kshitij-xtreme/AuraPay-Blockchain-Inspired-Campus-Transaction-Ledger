# Changelog

All notable changes to AuraPay will be documented in this file.

## [1.0.0] - 2026-05-04

### Added
- Initial release of AuraPay
- Blockchain-inspired transaction ledger system
- User account management with role-based access control
- Transaction creation and validation
- Block mining with Proof of Work
- Security utilities (SHA-256, HMAC, PBKDF2, RSA)
- Transaction pool management
- REST-like API endpoints
- Unit tests for core functionality
- Interactive CLI interface
- Comprehensive documentation
- Configuration management

### Features
- ✅ SHA-256 hashing for block integrity
- ✅ Merkle tree transaction verification
- ✅ User roles (STUDENT, STAFF, FACULTY, ADMIN, MERCHANT)
- ✅ Transaction status tracking
- ✅ Blockchain validation
- ✅ Transaction history
- ✅ Balance management
- ✅ Mining difficulty adjustment
- ✅ System statistics and monitoring
- ✅ Data persistence support

### Security
- ✅ PBKDF2 password hashing with salt
- ✅ HMAC-SHA256 transaction signing
- ✅ RSA digital signatures
- ✅ Input validation and sanitization
- ✅ Role-based access control
